# 📊 Análise de Dados Musical - SQL

Este projeto consiste numa análise exploratória de dados (EDA) de uma plataforma de streaming de música, utilizando **SQL (PostgreSQL/MySQL)** para extrair insights sobre o catálogo e o desempenho de faixas e álbuns.

## 📁 Estrutura do Repositório
- `analise-dados-sql-musica.sql`: Script completo com consultas para limpeza de dados, contagem e estatísticas descritivas.
- `albuns.csv`, `artistas.csv`, `faixas.csv`, `streams_mensais.csv`: Conjuntos de dados brutos utilizados na análise.
- `Relatório de Análise Musical.pdf`: Documentação detalhada com as conclusões da análise.

## 🚀 Consultas Implementadas
O script SQL aborda os seguintes pontos:
1. **Contagem Total:** Visão geral do volume de dados (total de faixas e álbuns únicos).
2. **Tratamento de Dados:** Verificação de valores nulos em colunas críticas como ID, Título e métricas de áudio.
3. **Estatísticas Descritivas:** Cálculo de médias, valores mínimos e máximos de duração e características das músicas.
4. **Ranking de Popularidade:** (Opcional) Identificação dos artistas e álbuns com maior volume de streams.

## 🛠️ Tecnologias Utilizadas
- **SQL:** Para manipulação e consulta de base de dados.
- **CSV:** Formato de armazenamento dos dados brutos.
- **GitHub:** Para versionamento de código e documentação do projeto.
